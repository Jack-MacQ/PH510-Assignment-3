PH510 Assignment 3: Numerical Solutions of Poisson and Laplace Equations 

Overview:

This project investigates numerical methods for solving the Poisson and Laplace equations in two dimensions. Two approaches are implemented:

 - A deterministic relaxation-based solver for grid-based solutions
 - A stochastic Monte Carlo random-walk method based on Green's functions
 
The aim is to compare these methods in terms of accuracy, convergence behaviour, and computational performance. In particular, the random-walk method is parallelised using MPI to explore scaling behaviour and efficiency.
